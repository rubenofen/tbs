# CSS

## Inline style

```html
<h1 style="color: red;">Hello World</h1>
```

## Tag style

```html
<style>
  h1 {
    color: red;
  }
</style>
```

## External style

```html
<link rel="stylesheet" href="style.css" />
```

## CSS Selectors

```css
/* Element selector */
h1 {
  color: red;
}

/* Class selector */
.title {
  color: red;
}

/* ID selector */
#title {
  color: red;
}

/* Attribute selector */
[type="text"] {
  color: red;
}

/* Pseudo-class selector */
a:hover {
  color: red;
}

/* Pseudo-element selector */
p::first-line {
  color: red;
}
```
